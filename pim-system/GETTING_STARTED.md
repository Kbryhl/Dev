# PIM System Development

This directory contains a full Product Information Management system built with Node.js/Express and React.

## Directory Structure

- `backend/` - Express.js server with SQLite database
- `frontend/` - React web application
- `README.md` - Full documentation
- `QUICKSTART.md` - Quick reference guide

## Immediate Next Steps

1. Open `QUICKSTART.md` for 5-minute setup guide
2. Read `README.md` for full documentation
3. Start the backend: `cd backend && npm install && npm start`
4. Start the frontend: `cd frontend && npm install && npm start` (new terminal)

## Key Features

✅ User authentication with JWT  
✅ Product CRUD operations  
✅ Flexible product fields & custom specs  
✅ Category management  
✅ Search & filtering  
✅ Role-based permissions  
✅ Responsive React UI  

## Tech Stack

- **Backend**: Node.js, Express, SQLite, JWT
- **Frontend**: React, Axios, React Router
- **Database**: SQLite (easily upgradeable to PostgreSQL)

All endpoints, configurations, and enhancement ideas are documented in the main README.md file.
